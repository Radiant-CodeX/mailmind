──────────────────────────────────────────────────────────────────────────────

PRODUCTION
================================================================================

Demo login works in production the same way.
Access: https://your-domain.com/api/demo/login

To remove later, just delete the account:
  DELETE FROM users WHERE id = (
    SELECT user_id FROM oauth_accounts
    WHERE email = 'demo@mailmind.app'
  );

================================================================================
TROUBLESHOOTING
================================================================================

"Demo account not found"
  → Run seed script: python scripts/seed_demo_account.py --db <DATABASE_URL>

"Page not loading"
  → Check backend is running: curl http://localhost:8000/api/health
  → Check frontend is running: curl http://localhost:3000

Can't click "Enter Demo" button
  → Check browser console for errors
  → Verify database has demo account: psql ... -c "SELECT * FROM oauth_accounts WHERE email='demo@mailmind.app';"
